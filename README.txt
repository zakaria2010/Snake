## Projet de snake 
"les commentaires sont inclus dans le code source"



Objectif: Faire un snake. On peut aussi jouer en mode 2 joueurs: chaque joueur à un serpent qu'il contrôle. 
Si un des serpents rencontre l'autre, le joueur correspondant perd.


Fonctionnalités attendues:
Déplacement avec des touches (flèches directionelles pour le premier joueur et w/a/s/d pour le second)
Apparition de points sur l'écran
Affichage du score du joueur
mode 2 joueurs


"les commentaires sont inclus dans le code source"

Mangez autant de pommes que possible.




## Controles


Utilisez les flèches pour contrôler votre serpent

(deuxième joueur : utilise w/a/s/d)
